3D Files

  • Game Boy Color Cart Cutting Guide (GBCCCG) - Two piece cutting guide for Game Boy Color cartridge shells. Can be printed on FDM 3D printers. Helps to align your shell cuts, saving time.

  • FDM Compatible Shell - Basic shell you can print at home with an FDM 3D printer. Adapted from this shell by krizdingus - as such these files are licensed CC BY-NC-SA 4.0. I recommend using a little tree support for the screw area. Uses a normal Game Boy cart screw.

  • Orange FM Shell by Beja - Beautiful resin printed shell. Will not print correctly on an FDM printer. Requires a M1.6 brass insert and M1.6x3mm screw. Created by Beja on the Orange GB discord!
