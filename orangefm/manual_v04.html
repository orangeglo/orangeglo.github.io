Orange FM - ROM Manual Version 0.4

ROM Version 0.4 is very similar to 0.3, so I will just detail what's different. See ROM Manual v0.3

New Features / Changes

  • Built-in Headphone Amp - If a headphone amp is installed on your cart, you can listen to the radio through the cart's combo antenna/headphone jack! Press B + Start to toggle the amp on and off. There is also an "Auto Amp" setting you can turn on, which will automatically turn the amp on when you plug in headphones.
  • Improved Manual Sync - Hold B to manually sync data. You can choose in the Data Sync settings which update mode to use, so you can use this feature to quickly sync RDS, etc.
  • Configurable De-Emphasis - Select between 75us and 50us de-emphasis constant, which affects the high frequency content. Different regions use different constants, but in general North America uses 75us and the rest of the world uses 50us.
  • Improved Stability - The ROM should almost never lock up now, and if it does, there is an improved error display overlay.
  • GBA Support (with amp) - If you have the headphone amp installed on your board, the radio will now boot up and be functional.
